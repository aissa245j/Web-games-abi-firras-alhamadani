Signal Heist

Overview

Infiltrate a secure facility by sequencing actions across a timeline. You must collect 3 data shards and exit before the alarm meter fills.

Facility Layout

Build a 3x3 grid of rooms. Each room has a security level and a reward. Patrol routes advance one room per round.

Planning Phase

  • Assign 4 action tokens to the next 2 rounds (move, hack, distract, hide).
  • Each hack rolls against the room security level. Failures raise the alarm meter.
  • Distractions move patrols; hiding reduces alarm by 1.

Win Condition

Collect 3 data shards and reach the Exit before the alarm meter hits 8. For a higher challenge, add a second patrol route.
