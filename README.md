# Simple Games Hub

A lightweight multi-game website built with plain HTML, CSS, and JavaScript. Each game lives in its own folder under `games/` with its own `index.html`, `style.css`, and `game.js` files.

## Structure

- `index.html` — main hub page with game cards and search
- `games/` — individual game folders

## Run

- Open `index.html` in a browser to play.
- Or try it live: `https://aissa245j.github.io/Web-games-abi-firras-alhamadani/index.html`.

## Add a Game

1. Create a new folder in `games/`.
2. Add `index.html`, `style.css`, and `game.js`.
3. Add a card to `index.html`.
